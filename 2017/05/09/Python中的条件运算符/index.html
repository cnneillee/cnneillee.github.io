Python中的条件运算符

2017-05-09
  • Scripts

一、概述

在 C、Java 等语言中，有一种常见的条件运算符，又叫“三目运算符”。
详情参见 条件运算符-wikipedia

?: 的一般形式为
<表达式1> ? <表达式2> : <表达式3>

二、Python 中的实现

Python 程序设计语言使用一种不同的语法处理条件表达式：
value_when_true if condition else value_when_false

这一特性在 Python 2.5 以前的版本并不可用。
