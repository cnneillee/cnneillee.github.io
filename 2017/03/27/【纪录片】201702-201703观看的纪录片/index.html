一、概述

大概从去年12月份就开始看纪录片了，前期看的不多。从20170206-至今，这段时间看的纪录片蛮多的。

我是在20170206乘动车来的学校，路上百无聊赖，便提前下好了不少纪录片以供观看。不曾想，实习的地铁上，这些纪录片也为我待了不少的兴致，陪伴我度过了拥挤的地铁时间。

三、细节

3.1 第一页

3.1.1 BBC.西西里：地中海奇境

风景蛮不错的

3.1.2 【央视纪录片】帝都咸阳

主要介绍的是秦始皇建国，秦始皇陵以及咸阳（西安）的一些东西。

3.1.3 湘钢故事：撑花牌 喜迎春

粤语，蛮有意思的。花牌作为香港市井市民里各种大事的宣传用品，经历了很长的年头，在老一辈心中留下了不可磨灭的印记，在不少人心里，也早就是香港的某个符号的象征。但随着现代都市生活发展的冲击，生存圈不断被压缩，也逐步走向没落。

3.1.4 中国诗词大会-武亦姝

非常有文采，居然还是高中生！佩服，才女！符合我个人对心灵伴侣的期待！

3.1.5 局座芳草地读者见面会全记录

局座，支持一下！主持人八面玲珑！虽然中间有一些小插曲，如铭牌、位子等没协调好，但是主持人真的是很灵活，化解了不少尴尬。
