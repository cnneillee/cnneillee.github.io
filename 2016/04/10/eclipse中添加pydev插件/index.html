Eclipse 中添加 PyDev 插件

2016-04-10
  • Scripts

一、思路

  • 启动 Eclipse,
  • 点击 Help->Install New Software…
  • 在弹出的对话框中，点 Add 按钮。
  • Name 中填: Pydev, Location 中填 http://pydev.org/updates
    …

按要求执行下一步操作，如遇错误，则重装。
若安装成功，则重启即可
