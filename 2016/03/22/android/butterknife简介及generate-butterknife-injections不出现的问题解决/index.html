一、概述

Butterknife 是一款 Android Studio 的功能强大插件。有了它，你几乎可以和 findViewById 说 byebye 了。

二、使用

Github 地址：https://github.com/avast/android-butterknife-zelezny

三、问题解决

Generate ButterKnife Injections 不出现的问题解决

  • 下载后重启须重启as，方生效

  • 打开 Gradle App，添加 compile 'com.jakewharton:butterknife:7.0.1' 依赖

  • 请务必将鼠标光标悬停在 R.layout.xxx 处，点击右键！请详细参阅 https://github.com/avast/android-
    butterknife-zelezny
