java环境搭建

  • JDK下载
  • JDK安装
  • JDK安装正确检查
  • java环境变量配置
  • java安装常见名词说明

#JDK下载

>进入oracle官网下载-->[oracle][oracle]
>直接点击进入java下载页面-->[Oracle Technology Network Java Java SE下载][javaseJDK]

#JDK安装

>安装包内找到可执行文件(jdkxxxx.exe)
>点击可执行文件，点选下一步[安装目录选择较为**纯净的目录**]
>安装目录很重要务必记住[拷贝在记事本中，安装结束后若环境变量未配置好，则要用到此目录]
>出现jre等工具选择下载时，可下也可不下

#JDK安装正确检查
cmd下，分别键入以下三个命令：

>java
>javac
>java -version
若无错误信息，则说明配置成功
若错误信息，则需配置环境变量

#java环境变量配置[未自动配置成功的情况下须完成此步骤]

>主要三个变量（后二者可基于第一个，设置为相对目录(%JAVA_HOME%目录)）
    - JAVA_HOME[可选，设置此目录作用是作为以下两个变量的参考目录]
        jdk的安装目录（例如D:\Java\jdk1.6.0_12）
    - CLASSPATH[指向java的类库，注意添加一个"."表示当前目录]
        `.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar`
    - PATH[指向java\javac可执行文件的目录]
        `*%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin*`
>也可只设置两个变量（但是这两个变量必须设置其绝对路径）
    - CLASSPATH[指向java的类库，注意添加一个"."表示当前目录]  
        `.;jdk安装目录\lib\dt.jar;jdk安装目录\lib\tools.jar`
    - PATH[指向java\javac可执行文件的目录]
        `*%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin*`
>[百度经验教程][javaenvironmentval]

#特别说明
环境变量：指向某个存放目标软件的目录
纯净的目录：无中文及中文字符、无空格，且目录的设置有特殊含义，按照一定的规范设置，便于查找及查看的目录
相对路径、绝对路径：

>类似地理的参考系，经纬度能够在某个星球范围内唯一确定某个点；
>而正前方100米的描述则只能在知道当前位置的情况下才能得到目标点
> 
>绝对路径：该计算机内能够唯一确定某个位置的路径
>相对路径：只有在已知参考目录之后，才能据此确定位置的路径
