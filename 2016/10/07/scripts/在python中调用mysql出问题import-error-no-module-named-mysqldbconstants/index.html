遇到如下异常：
File “C:\Users\Neil\PycharmProjects\ScrapyDouban\book\book\database.py”, line
4, in

解决方法：
从 https://pypi.python.org/pypi/MySQL-python/1.2.5 中下载 MySQL-
python-1.2.5.win32-py2.7.exe 并运行即可

上面只有32位的，下面这个链接里有32/64bit
http://www.codegood.com/archives/129

which was not found in the registry问题，解决方法见
http://blog.csdn.net/zklth/article/details/8117207
